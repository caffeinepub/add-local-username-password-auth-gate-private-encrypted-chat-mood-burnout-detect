# Specification

## Summary
**Goal:** Rename the landing page’s final CTA from “For Organizations” to “Subscription plans” and add a new Subscription Plans page accessible via client-side navigation.

**Planned changes:**
- Update the landing page Final CTA secondary button label to “Subscription plans”.
- Make the “Subscription plans” CTA navigate (without full page reload) to a new Subscription Plans page/screen.
- Add a Subscription Plans page that displays three plans (₹200, ₹800, ₹1600) with their session/time details and a prominent note: “The first session is entirely free.”
- Provide a clear way to return from the Subscription Plans page back to the landing page.
- Ensure the existing “Start MindVault” / experience flow remains unchanged.

**User-visible outcome:** Users can click “Subscription plans” on the landing page to view subscription pricing and the free first-session note, then easily return to the landing page.
